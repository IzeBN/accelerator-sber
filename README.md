# СберСтарт — Студенческий финансовый продукт

Конкурсный проект. Полнофункциональное демо студенческого пакета Сбербанка с умным помощником Степаном.

## Быстрый старт

```bash
docker-compose up --build
```

Открыть в браузере: **http://localhost**

## Стек

- **Frontend**: React 18 + TypeScript + Vite → nginx
- **Backend**: Python 3.11 + FastAPI + asyncpg + SQLAlchemy 2
- **БД**: PostgreSQL 16
- **Деплой**: Docker + docker-compose

## Функционал

- 🏠 **Лендинг** — промо-страница с описанием продукта
- 🎓 **Онбординг** — 4 экрана регистрации и настройки
- 📋 **Опросник** — 5 шагов настройки помощника Степана
- 📊 **Дашборд** — баланс, советы, аналитика трат
- 🏦 **Копилка** — цели накопления со стратегиями
- 📈 **Инвестиции** — упрощённый вход от 100₽
- 🎁 **Льготы** — все бонусы и кэшбэк в одном месте
- 👪 **Родители** — переводы без контроля трат
- ⚙️ **Админ** — модерация справок и метрики продукта
